  • 배송비는 얼마인가요?
    무료 배송비 혜택은 주문 건 기준 금액이 50,000원 이상부터 적용되며 50,000원 미만일 경우 배송비 3,000원이 부과됩니다.
    제주 지역은 5,000원 제주 외 도서지역은 5,000원이 추가 부과됩니다.
    출고는 주문번호를 기준으로 이루어지며 주문을 나누어 각 상품마다 주문번호가 다른경우 각각 출고되기 때문에 개별로 배송비가 부과됩니다.
    개별 주문 후 합배송 요청 시 처리가 불가능하오니 두 가지 이상 주문하게 되실 경우, 장바구니에 담으신 뒤 한 번에 결제해 주시기 바랍니다.
  • 주문상품 중 일부 상품만 취소하고 싶어요.
    결제 완료 이후 상품준비중 / 배송준비중 단계에서만 취소가 가능합니다.
    단, 배송준비중 단계에서 주문 확인 마감 시(오전 9시 30분, 오후2시)에는 처리가 불가한 점 양해 부탁드립니다.
    부분 취소 요청 시 고객센터에서 확인 후 처리를 도와드리고 있어 시간이 다소 소요될 수 있는 점 참고 부탁드립니다.
    ※ 부분 취소하여 무료 배송비 혜택이 파기되는 경우, 초도배송비는 환불금액에서 차감 후 정산 됩니다.
  • 출고 전 상품 변경은 어떻게 하나요?
    결제 완료 후 욥션 변경이 불가한 시스템으로 번거로우시겠지만 주문 취소 후 재구매를 부탁드립니다.
    주문 취소 시 상품준비중 / 배송준비중 단계에서만 취소가 가능합니다. 단, 배송준비중 단계에서 주문 확인 마감 시에는 처리가 불가한 점 양해 부탁드립니다.
    상품준비중 단계에서 주문건 전체 취소 시 즉시 취소처리 완료되며 부분 취소 요청 시 고객센터에서 확인후 처리를 도와드리고 있어 시간이 다소 소요될 수 있는 점 참고 부탁드립니다.
    주문 확인 마감 후 변심으로 인한 교환 / 반품 시 최초 배송비와 반품 배송비는 고객님 부담이므로 신중하게 구매해 주시기 바랍니다.

    단, 주문 확인 마감 시(오전9시 30분, 오후 2시)에는 처리가 불가한 점 양해 부탁드립니다.
  • 주문 취소는 어떻게 하나요?
